```

app.json             # App configuration

├── User.js           # User schema
├── Product.js        # Product schema
└── Order.js          # Order schema

routes/
├── authRoutes.js     # Authentication
├── productRoutes.js  # Products
```

## 📊 Database Schema

### Users Collection

- Name, Email, Password (hashed)
- Role (user/admin)
- Phone, Address (street, city, state, country, zipCode)
- Created at, Updated at

### Products Collection

- Name, Description, Price
- Category, Stock, Rating
- Images (URL, alt text)
- Brand, SKU
- Reviews
- Created at, Updated at

### Orders Collection

- User ID (reference)
- Products (array with details)
- Total Price
- Shipping Address
- Status, Payment Status, Payment Method
- Tracking Number
- Created at, Updated at
